# GESO-requirements

#### https://github.com/VDobranov/GESO-requirements

Требования к атрибутивному наполнению IFC-моделей Государственной экспертизы Свердловской области в формате Markdown.

Приложение №4 к ТИМ-стандарту 2.0 (Приказ № 200-п от 27.04.2024, стр. 38–102) переведено из PDF в 39 Markdown-файлов. Охватывает разделы АР, КР и ИС: архитектурные решения (стены, перегородки, двери, окна, кровля, помещения), конструктивные решения (несущие стены, перекрытия, колонны, балки, фундамент, лестницы), инженерные сети (трубопроводы, воздуховоды, электрооборудование, освещение).

Цель репозитория — предоставить структурированный источник данных для автоматизации проверки и заполнения атрибутов IFC-моделей с использованием LLM.
