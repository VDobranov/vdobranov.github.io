# ANCHOR-BOLT-GENERATOR

#### https://vdobranov.github.io/ANCHOR-BOLT-GENERATOR/

Веб-приложение для генерации анкерных (фундаментных) болтов по ГОСТ 24379.1-2012 с экспортом в IFC.

Поддерживает типы болтов 1.1, 1.2, 2.1, 5; диаметры М12–М48; марки стали 09Г2С, ВСт3пс2, 10Г2. Настраивается тип геометрии в IFC, класс сборки (IfcMechanicalFastener / IfcElementAssembly) и набор подключаемых PropertySet — в том числе специализированные pset для прохождения российской экспертизы. Результат отображается в 3D-просмотрщике прямо в браузере.
