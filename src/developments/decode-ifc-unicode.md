# DECODE-IFC-UNICODE

#### https://vdobranov.github.io/DECODE-IFC-UNICODE/

Утилита для декодирования Unicode-escape-последовательностей из IFC-файлов в читаемый текст.

Вставьте строку вида `\X2\…\X0\` — приложение немедленно вернёт декодированный результат. Удобно при ручном анализе содержимого IFC или отладке экспорта из САПР с нелатинскими символами.
